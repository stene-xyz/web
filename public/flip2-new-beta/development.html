Root + ADB

This guide assumes you're on Linux. If not, create a bootable USB stick and do the jailbreak from there.

Part 1: Preparation

  1. Download the following files:
    mtkclient
    patched boot.img - place this in the mtkclient folder. TODO: add file
  2. Install fastboot, libusb, and fuse2 from your distro's package manager. Also make sure Python is up to date.
  3. In the mtkclient folder, run "python3 -m pip install -r requirements.txt"
  5. Backup your device by running "mkdir backup; sudo python3 mtk.py rl backup" - this will take a while but it is important! Copy backup files elsewhere if running from a Live USB stick.

Part 2: Jailbreaking

  1. Reboot the device into fastboot mode by running "sudo python3 mtk.py payload --metamode FASTBOOT"
  2. Unlock the device for flashing with "fastboot flashing unlock" - this will erase all user data. Advanced users can run "sudo python3 mtk.py da seccfg unlock" to prevent a wipe, but I haven't tested it very much and it may cause issues.
  3. Flash the patched boot image with "fastboot flash boot boot.img"
  4. Reboot the device and complete initial setup (if required)
  5. Open the app list by entering "*#*#217703#*#*" on the home screen
  6. Launch Magisk and let it do the final setup
  8. Allow it to reboot your device
  9. Open Magisk again and dismiss the popup
  10. Open the browser and click this link to download adbio
  11. Open your notifications and click the downloaded file
  12. Allow the browser to install apps, then install adbio
  13. Use adbio to enable ADB. TODO: in-depth
